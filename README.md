# Dungeon Crawler

Site de um jogo dungeon crawler. Feito primariamente para estudar Array 2D, e classes de JS.

## Tecnologias

- React
- JavaScript
- Typescript

## Escopo do projeto

### Página principal

Página única(até o momento) onde o usuário realiza todas as interações(batalhas, equipa itens, usa magias, e etc)
